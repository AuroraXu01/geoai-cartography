# Styles

Basemaps:

- `none`: no basemap.
- `naturalearth`: bundled land and country boundaries.
- `naturalearth-muted`: softer land, lighter boundaries.
- `naturalearth-gray`: grayscale publication background.
- `naturalearth-dark`: dark background for presentations.
- `online`: online XYZ/contextily tiles.

North arrows:

- `simple`: arrow with N label.
- `triangle`: filled triangle with N label.
- `compass`: four-point compass.
- `none`: omit north arrow.

Scale bars:

- `line`: single heavy line with ticks.
- `alternating`: two-segment black/white bar.
- `ticks`: thin line with end ticks.
- `none`: omit scale bar.

Color aliases:

- Sequential: `geo_seq`, `water`, `terrain_soft`, `risk_orange`, `risk_red`, `green`
- Diverging: `geo_div`, `residual`, `change`
- Categorical: `geo_cat`, `landuse`, `tab20`, `Set2`, `Dark2`

Layouts:

- `single`: one map or image.
- `grid`: equal-sized panels in rows/columns.
- `main-sub`: one large main image plus smaller subpanels.
- `side-by-side`: two or more equal comparison panels.
- `main-inset`: render a main map with an overview inset when supported by the map script.
