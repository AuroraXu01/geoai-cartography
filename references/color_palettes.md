# Color Palettes

Recommended Matplotlib color maps:

- Sequential: `viridis`, `YlGnBu`, `YlOrRd`, `cividis`, `magma`
- Diverging: `RdBu_r`, `PiYG`, `BrBG`, `coolwarm`
- Categorical: `tab10`, `tab20`, `Set2`, `Dark2`

Guidance:

- Use sequential color maps for ordered magnitudes.
- Use diverging color maps when values have a meaningful center, such as anomaly, residual, or change.
- Use categorical palettes for unordered land-use, region, or class labels.
- For journal figures, prefer `viridis`, `cividis`, `YlGnBu`, or carefully selected categorical palettes.
